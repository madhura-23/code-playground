# 🧠 ML-Experiments

A personal collection of hands-on Machine Learning experiments covering core algorithms — from scratch implementations to real-world datasets. Built for learning, reference, and exploration.

---

## 📁 Repository Structure

```
ML-Experiments/
├── Backpropogation.ipynb               # Neural network training via backprop
├── Descision_Tree.ipynb                # Decision Tree classifier
├── Ensemble.ipynb                      # Ensemble methods (Bagging, Boosting, etc.)
├── KMEANS.ipynb                        # K-Means clustering (unsupervised)
├── Logistic_Regression.ipynb           # Logistic Regression classifier
├── NB2.ipynb                           # Naive Bayes classifier
├── iris.ipynb                          # Classic Iris dataset exploration & classification
├── knn.ipynb                           # K-Nearest Neighbors
├── linrear_regresion.ipynb             # Linear Regression
├── movie_recommendation_system.ipynb   # Collaborative filtering recommendation system
├── Movie rating/                       # Movie ratings dataset & analysis
├── datasets/                           # Raw datasets used across experiments
├── notebooks/                          # Additional/supporting notebooks
├── User_Data.csv                       # Sample user dataset

```

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/madhura-23/code-playground.git
cd code-playground/ML-Experiments
```

### 2. Install dependencies

```bash
pip install numpy pandas matplotlib seaborn scikit-learn jupyter
```

### 3. Launch Jupyter

```bash
jupyter notebook
```

Then open any `.ipynb` file to explore an experiment.
